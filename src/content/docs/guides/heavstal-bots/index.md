# Introduction to Heavstal Bots Platform
- Name: Heavstal Bots
- Link (url): https://bots.heavstal.com.ng

1. **What is Heavstal Bots Platform?:**

[Heavstal Bots Platform](https://bots.heavstal.com.ng) is a platform designed for one main goal: **Heavstal Authentication Code (AuthCode) generation and management**

2. **What is an Heavstal AuthCode?:**

The Heavstal AuthCode (or more preferably *AuthCode*) is a 46 digits long string used to authenticate a user using Heavstal techs third party bots like verselor, it is used to grant useds access to features, it's also used to grant a user premium tag and therefore access to premium only features.
